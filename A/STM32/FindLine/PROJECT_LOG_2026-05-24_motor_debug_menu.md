# PD42S1 电机调试菜单日志

## 修改时间

2026-05-24

## 背景

为了后续调试云台底座，需要在主界面增加一个手动电机调试入口。进入调试菜单后，通过板载按键直接控制 PD42S1 电机顺时针或逆时针小步运动。

## 本次修改

- 在主菜单新增 `Motor Debug` 选项。
- 进入 `Motor Debug` 时初始化 PD42S1、使能电机、清除状态并清零当前位置。
- 在调试界面中：
  - PC3 按键控制电机顺时针运动一步。
  - PC2 按键控制电机逆时针运动一步。
  - PC0 返回主菜单，并停止、清状态、失能电机。
- 当前单步参数：
  - 步进角度：5 度。
  - 转速：60 RPM。
  - 加减速度：20。
- OLED 显示当前阶段、按键方向、单步角度、速度参数、累计调试角度和最近一次返回结果。

## 验证

- 已执行 `cmake --build build\Debug`，构建通过。
- 构建结果：RAM 使用 3376 B，FLASH 使用 27620 B。

## 后续调整

如需修改调试手感，可在 `Core/Src/main.c` 中调整 `APP_MOTOR_DEBUG_STEP_DEG`、`APP_MOTOR_DEBUG_RPM`、`APP_MOTOR_DEBUG_ACCEL`。
